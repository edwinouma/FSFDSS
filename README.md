# Food Security Forecasting and Decision Support System (FSFDSS)

## Overview

FSFDSS is a web-based machine learning platform developed as part of a Master's research project to forecast food security outcomes in Somalia.

The system integrates:

- Historical monitoring
- XGBoost forecasting
- SHAP model interpretation
- Scenario analysis
- Automated situation reporting

## Food Security Indicators

- Food Consumption Score (FCS)
- Reduced Coping Strategy Index (rCSI)
- Household Hunger Scale (HHS)

## Technologies

- Python
- Streamlit
- XGBoost
- SHAP
- Plotly
- Pandas
- Scikit-learn
